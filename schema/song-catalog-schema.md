# Song Catalog Schema

Each song should eventually have one record with the following fields.

## Core Song Fields

- song_id
- title
- alternate_titles
- lyricist
- status
- project
- album_or_collection
- genre
- mood
- themes
- lyrics
- suno_url
- soundcloud_url
- youtube_url
- tiktok_url
- notes

## Image / Artwork Fields

- primary_cover_image
- youtube_image
- tiktok_image
- image_prompt
- image_notes

## Publishing Fields

- release_status
- release_date
- copyright_notes
- credits
- description
- hashtags

## Internal Tracking

- created_date
- last_updated
- version_notes
