# Hooley the Poet Archive Plan

Purpose:
Create a permanent archive of songs, lyrics, images, prompts, playlists, videos, and publishing materials for Hooley the Poet.

Long-Term Goals:
- HooleyThePoet.com website
- Searchable song catalog
- Songbooks and printed collections
- SoundCloud and YouTube publishing support
- Artwork archive
- Prompt archive
- Future AI-assisted publishing tools

Technology:
- GitHub repository
- Cloudflare Pages
- Supabase database
- Supabase Storage or Cloudflare R2

Status:
Initial repository structure created June 2026.
